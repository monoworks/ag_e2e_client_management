## やり取り 3: 確認事項への回答（13:31）

### ユーザー（kenji）:
> 1: ユーザー間で共有したいので、jsonファイルへの保存とか可能でしょうか？
> 2: メモ帳で作成したテキストのみで大丈夫です。
> 3: 画面構成はいったん提示いただいた内容でお願いします。
> 4: 先々変更するかもしれませんが、ステータスは頂いた内容でお願いします。

### Antigravity:
フィードバックを受けて実装計画を更新:
- **データ永続化を変更**: localStorage → **GitHub API + リポジトリ内JSONファイル**
  - リポジトリの `data/` フォルダに `clients.json`, `projects.json` 等を保存
  - GitHub REST API で読み書き、SHAベースの楽観的ロックで競合検知
  - ユーザーは GitHub Personal Access Token を設定画面で入力
- **議事メモ**: テキストファイル（.txt）のドラッグ&ドロップアップロードに対応
- **画面構成**: 提案通り＋**設定画面**（GitHub Token設定用）を追加
- **ステータスフロー**: 提案通り（見込み→提案中→交渉中→成約/失注→深耕）
